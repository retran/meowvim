## рЯТ° Tips

- Leader key is `<space>`
- Use `<leader>ohk` to search all keymaps interactively
- Most keymaps work in visual mode too
- Flash jump (`<leader><space>`) is your friend for quick navigation
- `*` marks Neovide-only bindings
